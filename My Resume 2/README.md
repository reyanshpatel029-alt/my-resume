# Reyansh Patel Portfolio

Professional static portfolio rebuild.

## Structure

- `index.html`: Main one-page portfolio.
- `about.html`: Secondary about page.
- `assets/css/styles.css`: Design tokens, layout, components, states, and responsive rules.
- `assets/js/main.js`: Theme toggle, mobile menu, scroll progress, reveal animation, counters, skill bars, copy email, and contact form behavior.

## Notes

- The design avoids decorative blobs, custom cursors, and heavy loading screens.
- The project uses system fonts for fast loading.
- JavaScript is deferred and written without dependencies.
- Motion respects `prefers-reduced-motion`.
- The profile image keeps the original local path from the previous portfolio; replace it with a project-local image later for best portability.
